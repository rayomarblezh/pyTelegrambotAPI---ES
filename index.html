Introducción

pyTelegramBotAPI es una biblioteca Python simple pero extensible para la API de Telegram Bot. Proporciona una interfaz sencilla y fácil de usar para interactuar con la API de Telegram.

Características principales

  • ✓ Soporte completo para la API de Telegram Bot
  • ✓ Fácil de usar y extender
  • ✓ Soporte para manejadores de comandos y mensajes
  • ✓ Manejo de teclados personalizados

Casos de uso

  • ✓ Bots de atención al cliente
  • ✓ Notificaciones automáticas
  • ✓ Integraciones con otros servicios
  • ✓ Juegos y quizzes interactivos

Instalación

Puedes instalar pyTelegramBotAPI usando pip, el gestor de paquetes de Python:

# Instalación básica
pip install pyTelegramBotAPI

# Instalación con soporte para API asíncrona
pip install pyTelegramBotAPI[async]

Requiere Python 3.7 o superior. Se recomienda usar un entorno virtual para la instalación.

Ejemplos básicos

1. Bot eco

Un bot simple que repite todo lo que le envíes:

import telebot

# Crea una instancia del bot con tu token
bot = telebot.TeleBot("TU_TOKEN_AQUÍ")

# Manejador para mensajes de texto
@bot.message_handler(func=lambda message: True)
def echo_all(message):
    bot.reply_to(message, message.text)

# Inicia el bot
bot.polling()

2. Bot con comandos

Un bot que responde a comandos específicos:

import telebot

bot = telebot.TeleBot("TU_TOKEN_AQUÍ")

# Comando /start
@bot.message_handler(commands=['start'])
def send_welcome(message):
    bot.reply_to(message, "¡Hola! Soy un bot de ejemplo.")

# Comando /help
@bot.message_handler(commands=['help'])
def send_help(message):
    bot.reply_to(message, "Comandos disponibles:\n/start - Inicia el bot\n/help - Muestra esta ayuda")

bot.polling()

3. Bot con teclado personalizado

Un bot que muestra un teclado personalizado:

import telebot
from telebot import types

bot = telebot.TeleBot("TU_TOKEN_AQUÍ")

# Crea un teclado personalizado
@bot.message_handler(commands=['opciones'])
def show_options(message):
    markup = types.ReplyKeyboardMarkup(row_width=2)
    item1 = types.KeyboardButton("Opción 1")
    item2 = types.KeyboardButton("Opción 2")
    item3 = types.KeyboardButton("Opción 3")
    markup.add(item1, item2, item3)
    
    bot.send_message(message.chat.id, "Elige una opción:", reply_markup=markup)

# Maneja las opciones seleccionadas
@bot.message_handler(func=lambda message: True)
def handle_message(message):
    if message.text == "Opción 1":
        bot.send_message(message.chat.id, "Elegiste la Opción 1")
    elif message.text == "Opción 2":
        bot.send_message(message.chat.id, "Elegiste la Opción 2")
    elif message.text == "Opción 3":
        bot.send_message(message.chat.id, "Elegiste la Opción 3")

bot.polling()

Referencia API

Método Descripción Ejemplo
send_message Envía un mensaje de texto bot.send_message(chat_id, text)
reply_to Responde a un mensaje específico bot.reply_to(message, text)
send_photo Envía una foto bot.send_photo(chat_id, photo)
send_document Envía un documento bot.send_document(chat_id, document)
edit_message_text Edita el texto de un mensaje bot.edit_message_text(new_text, chat_id, message_id)
delete_message Elimina un mensaje bot.delete_message(chat_id, message_id)
send_chat_action Muestra acción del bot (escribiendo, grabando voz, etc.) bot.send_chat_action(chat_id, action)
forward_message Reenvía un mensaje bot.forward_message(to_chat_id, from_chat_id, message_id)
send_audio Envía un audio bot.send_audio(chat_id, audio)
send_voice Envía un mensaje de voz bot.send_voice(chat_id, voice)
send_video Envía un video bot.send_video(chat_id, video)
send_location Envía una ubicación bot.send_location(chat_id, latitude, longitude)
send_contact Envía un contacto bot.send_contact(chat_id, phone_number, first_name)
send_sticker Envía un sticker bot.send_sticker(chat_id, sticker)
answer_callback_query Responde a una consulta de callback bot.answer_callback_query(callback_query_id)
edit_message_reply_markup Edita el markup de un mensaje bot.edit_message_reply_markup(chat_id, message_id, reply_markup)
get_chat Obtiene información de un chat bot.get_chat(chat_id)
get_chat_member Obtiene información de un miembro del chat bot.get_chat_member(chat_id, user_id)
get_chat_administrators Obtiene los administradores de un chat bot.get_chat_administrators(chat_id)
get_chat_members_count Obtiene el número de miembros en un chat bot.get_chat_members_count(chat_id)

Tipos de manejadores

message_handler

Maneja mensajes basados en diferentes criterios:

  • commands=['start', 'help']
  • func=lambda m: True
  • content_types=['text', 'photo']

callback_query_handler

Maneja interacciones con botones inline:

  • func=lambda call: True
  • Manejo de datos en call.data
